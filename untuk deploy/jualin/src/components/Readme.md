# Folder: components/

Berisi komponen tampilan (UI). Tidak mengandung logika bisnis.

## Subfolder:
- ui → Elemen UI seperti Button, Input, Card, dll.
- auth → Komponen spesifik tampilan autentikasi seperti LoginForm & RegisterForm.

## Prinsip:
- Komponen di sini hanya bertanggung jawab pada UI.
- Semua logic dipanggil dari modules/auth.
